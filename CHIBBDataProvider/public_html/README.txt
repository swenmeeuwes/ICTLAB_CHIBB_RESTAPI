Description
-----------------
This folder contains a 'data provider' for the CHIBB project, 
which is a script that posts semi-realistic values to the webservice with the use of Perlin Noise and values ranges.

This was a solution to the problem that there were no sensors available in the CHIBB house yet.
This application is used to fill the database with sample data.

Running
-----------------
The system can be ran be called the 'npm start' command in the same folder as the 'package.json' file.
NOTE: If the node_modules folder is missing run the 'npm install' command beforehand to install all dependencies
